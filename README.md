## 🔎 Visão Geral

O **MercadoFicticio** é um protótipo de aplicação back‑end para demonstrar operações CRUD em um cenário de supermercado:

- **Produtos**: cadastro, listagem, atualização e remoção.  
- **Clientes**: gerenciamento de dados de clientes.  
- **Pedidos**: criação de pedidos com múltiplos itens, cálculo de totais e histórico.

---

## ✨ Funcionalidades

- Cadastro e gerenciamento de **Produtos** (nome, preço, estoque).  
- Cadastro e gerenciamento de **Clientes** (nome, e‑mail, telefone).  
- Criação de **Pedidos** vinculados a um cliente, com data, itens e valor total.  
- Validações básicas: estoque mínimo, campos obrigatórios.  
- Rotas RESTful estruturadas e documentadas.
